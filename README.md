## NeoVIM config by Hamilton Chevez

### Dependencies
Git
Nvim
a C compiler (mingw for windows, gcc for unix)
fzf
make
cmake


### Setup
#### Windows
- Install chocolatey
- Run `choco install mingw` in an admin powershell
- Run `refreshenv` or open a new shell
- Start Neovim

choco install make
choco install fzf
choco install mingw
choco install cmake
